# RAG-Assistant-VN

A production-oriented **Vietnamese-English Retrieval-Augmented Generation (RAG) assistant** designed for enterprise knowledge search and question answering across internal documents.

This project enables users to upload and query multi-format documents such as **PDF, DOCX, TXT, Markdown, CSV, and FAQ datasets**, then receive accurate, grounded answers with source citations.

Built to simulate real-world business use cases such as:

* Internal knowledge base assistant
* HR / policy document assistant
* SOP and operations search assistant
* Product / support documentation copilot
* Vietnamese-English bilingual enterprise search

## Key Features

* 🇻🇳🇬🇧 **Vietnamese-English bilingual retrieval**
* 📄 Multi-document ingestion pipeline
* 🔍 Semantic search with vector database
* 📚 Citation-grounded answers
* ⚡ FastAPI backend API
* 🧠 LLM-powered answer generation
* 📊 Query logging, latency, and cost tracking
* 🧪 Retrieval and answer quality evaluation
* 🐳 Dockerized deployment
* 🔄 CI/CD ready structure

## Tech Stack

* Python
* FastAPI
* PostgreSQL + pgvector / Qdrant
* OpenAI / Gemini / local embedding models
* Docker
* GitHub Actions
* Streamlit / Next.js (optional UI)
